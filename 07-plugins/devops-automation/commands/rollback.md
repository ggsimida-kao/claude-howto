# Rollback Deployment（回滚部署）

Rollback to previous stable version:

回滚到先前稳定版本：

1. Identify previous deployment（识别先前部署）
2. Verify rollback target is healthy（验证回滚目标是健康的）
3. Execute rollback procedure（执行回滚程序）
4. Run health checks（运行健康检查）
5. Notify team（通知团队）
