# System Status Check（系统状态检查）

Check system health across all services:

检查所有服务的系统健康状态：

1. Query Kubernetes pod status（查询 Kubernetes Pod 状态）
2. Check database connections（检查数据库连接）
3. Monitor API response times（监控 API 响应时间）
4. Review error rates（审查错误率）
5. Check resource utilization（检查资源利用率）
6. Report overall health（报告整体健康状态）
