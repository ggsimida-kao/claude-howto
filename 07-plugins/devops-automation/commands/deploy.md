# Deploy Application（部署应用程序）

Execute deployment workflow:

执行部署工作流：

1. Run pre-deployment checks（运行部署前检查）
2. Build application（构建应用程序）
3. Run tests（运行测试）
4. Deploy to target environment（部署到目标环境）
5. Run health checks（运行健康检查）
6. Notify team on Slack（在 Slack 上通知团队）
